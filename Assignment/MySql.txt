============================================================
                 MYSQL NOTES
         DATABASE OPERATIONS & SQL COMMANDS
============================================================

Definition
----------
MySQL is an open-source Relational Database Management System
(RDBMS) used to store, manage, retrieve, and manipulate data.

Database
--------
A database is an organized collection of related data.

Example

College Database

Student Table

Faculty Table

Course Table

============================================================
TYPES OF SQL COMMANDS
============================================================

1. DDL (Data Definition Language)

Used to create and modify database objects.

Commands

CREATE

ALTER

DROP

TRUNCATE

RENAME

------------------------------------------------------------

2. DML (Data Manipulation Language)

Used to insert, update and delete records.

Commands

INSERT

UPDATE

DELETE

------------------------------------------------------------

3. DQL (Data Query Language)

Used to retrieve data.

Command

SELECT

------------------------------------------------------------

4. DCL (Data Control Language)

Used for permissions.

Commands

GRANT

REVOKE

------------------------------------------------------------

5. TCL (Transaction Control Language)

Used to manage transactions.

Commands

COMMIT

ROLLBACK

SAVEPOINT

============================================================
DATABASE OPERATIONS
============================================================

1. Show All Databases
---------------------

Command

SHOW DATABASES;

Purpose

Displays all databases available in MySQL Server.

------------------------------------------------------------

2. Create a Database
--------------------

Command

CREATE DATABASE college;

Purpose

Creates a new database named "college".

------------------------------------------------------------

3. Create Database Only If It Doesn't Exist
-------------------------------------------

Command

CREATE DATABASE IF NOT EXISTS college;

Purpose

Prevents an error if the database already exists.

------------------------------------------------------------

4. Select (Use) a Database
--------------------------

Command

USE college;

Purpose

Makes "college" the active database.

------------------------------------------------------------

5. Show Current Database
------------------------

Command

SELECT DATABASE();

Purpose

Displays the currently selected database.

------------------------------------------------------------

6. Delete a Database
--------------------

Command

DROP DATABASE college;

Purpose

Deletes the database permanently.

------------------------------------------------------------

7. Delete Database Only If It Exists
------------------------------------

Command

DROP DATABASE IF EXISTS college;

Purpose

Deletes the database only if it exists.

============================================================
TABLE OPERATIONS
============================================================

1. Show All Tables
------------------

Command

SHOW TABLES;

Purpose

Displays all tables in the selected database.

------------------------------------------------------------

2. Create a Table
-----------------

Command

CREATE TABLE student(
id INT,
name VARCHAR(50),
age INT
);

Purpose

Creates a new table named "student".

------------------------------------------------------------

3. Describe Table Structure
---------------------------

Command

DESC student;

OR

DESCRIBE student;

Purpose

Displays table structure including columns, data types,
null values and keys.

------------------------------------------------------------

4. View Table Creation Statement
--------------------------------

Command

SHOW CREATE TABLE student;

Purpose

Shows the SQL statement used to create the table.

------------------------------------------------------------

5. Rename a Table
-----------------

Command

RENAME TABLE student TO students;

Purpose

Changes the table name.

------------------------------------------------------------

6. Delete a Table
-----------------

Command

DROP TABLE student;

Purpose

Deletes the table permanently.

------------------------------------------------------------

7. Delete Table Only If It Exists
---------------------------------

Command

DROP TABLE IF EXISTS student;

Purpose

Deletes the table only if it exists.

============================================================
ALTER TABLE OPERATIONS
============================================================

1. Add a New Column
-------------------

Command

ALTER TABLE student
ADD email VARCHAR(100);

Purpose

Adds a new column to the table.

------------------------------------------------------------

2. Modify Column Data Type
--------------------------

Command

ALTER TABLE student
MODIFY age BIGINT;

Purpose

Changes the data type of a column.

------------------------------------------------------------

3. Rename a Column
------------------

Command

ALTER TABLE student
RENAME COLUMN name TO student_name;

Purpose

Changes the column name.

------------------------------------------------------------

4. Drop a Column
----------------

Command

ALTER TABLE student
DROP COLUMN email;

Purpose

Deletes a column from the table.

============================================================
INSERT OPERATIONS
============================================================

1. Insert One Record
--------------------

Command

INSERT INTO student
VALUES(101,'Rahul',20);

Purpose

Inserts one row into the table.

------------------------------------------------------------

2. Insert Specific Columns
--------------------------

Command

INSERT INTO student(id,name)
VALUES(102,'Kiran');

Purpose

Inserts values into selected columns only.

------------------------------------------------------------

3. Insert Multiple Records
--------------------------

Command

INSERT INTO student
VALUES
(103,'Teja',21),
(104,'Ravi',22),
(105,'Sita',20);

Purpose

Inserts multiple rows at once.

============================================================
SELECT OPERATIONS
============================================================

1. Display All Records
----------------------

Command

SELECT * FROM student;

Purpose

Displays all rows and columns.

------------------------------------------------------------

2. Display Selected Columns
---------------------------

Command

SELECT id,name FROM student;

Purpose

Displays only selected columns.

------------------------------------------------------------

3. Display Unique Values
------------------------

Command

SELECT DISTINCT age FROM student;

Purpose

Shows unique values only.

============================================================
UPDATE OPERATIONS
============================================================

Command

UPDATE student
SET age=21
WHERE id=101;

Purpose

Updates existing records.

============================================================
DELETE OPERATIONS
============================================================

Command

DELETE FROM student
WHERE id=101;

Purpose

Deletes selected records.

============================================================
TRUNCATE TABLE
============================================================

Command

TRUNCATE TABLE student;

Purpose

Deletes all records from the table but keeps the table
structure.

============================================================
DROP TABLE
============================================================

Command

DROP TABLE student;

Purpose

Deletes the entire table including its structure.

============================================================
DIFFERENCE
============================================================

DELETE

Removes selected rows.

Table remains.

Can use WHERE.

Can rollback (with transactions).

------------------------------------------------------------

TRUNCATE

Deletes all rows.

Table remains.

Cannot use WHERE.

Faster than DELETE.

------------------------------------------------------------

DROP

Deletes complete table.

Table structure removed.

Cannot recover easily.

COMMON DATA TYPES
============================================================

INT

Example

100

------------------------------------------------------------

BIGINT

------------------------------------------------------------

VARCHAR(n)

Variable-length text

Example

VARCHAR(50)

------------------------------------------------------------

CHAR(n)

Fixed-length text

------------------------------------------------------------

FLOAT

Decimal numbers

------------------------------------------------------------

DOUBLE

Large decimal numbers

------------------------------------------------------------

DATE

Stores date

Example

2026-07-20

------------------------------------------------------------

TIME

Stores time

------------------------------------------------------------

DATETIME

Stores both date and time.

============================================================
CONSTRAINTS
============================================================

PRIMARY KEY

Unique identifier.

------------------------------------------------------------

NOT NULL

Cannot store NULL.

------------------------------------------------------------

UNIQUE

Duplicate values not allowed.

------------------------------------------------------------

DEFAULT

Assigns a default value.

------------------------------------------------------------

AUTO_INCREMENT

Automatically increases numeric values.
